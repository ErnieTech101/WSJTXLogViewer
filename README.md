# WSJT-X Log Viewer (ASCII TUI + QRZ)

Command-line 'Terminal User Interface' utility to view WSJT-X ADIF logs, look up callsigns via **QRZ XML**, and upload QSOs to your **QRZ Logbook**. No GUI, just a fast and simple way to access your WSJT-X logbook without the need for a separate logbook.

## Features
- Sticky header + keybar (pure ASCII; no GUI)
- QRZ XML lookup (name, QTH, grid, DXCC)
- QRZ Logbook upload (INSERT, optional REPLACE)
- Works great on Windows/macOS/Linux terminals
